## Phase 1: Perron uniqueness `srgM_k_eq_one` (commit `8ee2621`)

### 当初の戦略 (失敗): spectral orthogonality

教科書の Hoffman-Singleton 証明は `OrthonormalBasis` 経由で:

```
e_i, e_j ∈ k-eigenspace ⟹ both proportional to 𝟙 (by perron_collinear)
                       ⟹ orthogonality forces 0 ⟹ contradiction with basis nonzero
```

しかし Lean では `EuclideanSpace ℝ W` の `OrthonormalBasis`, `WithLp.ofLp`, inner product 経由が重く, formalize は数百行レベルの作業.

### 採用した戦略: 純代数

代わりに 4 つの identity を組み合わせる純代数で m_k = 1 を導出:

```
(I)  sum:        m_k + m_+ + m_- = k² + 1
(II) trace:      m_k k + m_+ λ_+ + m_- λ_- = 0
(III) trace-sq:  m_k k² + m_+ λ_+² + m_- λ_-² = (k² + 1) k
(IV) quadratic:  λ_±² + λ_± - (k - 1) = 0
```

(IV) で λ_+², λ_-² を消去 ⟹ (II) で m_+ λ_+ + m_- λ_- 消去 ⟹ (I) で m_+ + m_- 消去 ⟹ `m_k(k² + 1) = k² + 1` 直接導出.

`linear_combination` で一発:

```lean
linear_combination
  hpart +
    (-((srgM_plus hHerm k : ℝ))) * hlp +
    (-((srgM_minus hHerm k : ℝ))) * hlm +
    htrA +
    (-((k : ℝ) - 1)) * hsum
```

(III) を得るために `trace(A²) = ∑ λ_i²` の bridge が必要だが, これも matrix calculation で sorry-free に:

```lean
private theorem trace_sq_eq_sum_eigenvalues_sq
    {A : Matrix W W ℝ} (hHerm : A.IsHermitian) :
    (A * A).trace = ∑ i, (hHerm.eigenvalues i)^2
```

証明: `IsHermitian.spectral_theorem` で `A = U D U⁻¹` (実は `conjStarAlgAut`),
`A * A = conjStarAlgAut U (D * D)` (`_root_.map_mul` で逆向き),
`trace_mul_cycle` で `U` を奥に, `Unitary.coe_star_mul_self` で `star U * U = 1`,
`diagonal_mul_diagonal` + `trace_diagonal` で対角和に帰着. ~25 行.

そして SRG 用 `trace(A²) = nk`:
```
A² + A - (k-1)I = J  (SRG identity)
⟹ A² = J - A + (k-1)I
⟹ tr(A²) = tr(J) - tr(A) + (k-1) tr(I) = n - 0 + (k-1) n = nk
```

純粋な matrix 等式操作 + `Matrix.trace_add/sub/smul/one` + `trace_adjMatrix = 0`.

**Key insight**: spectral 経路を諦め, dichotomy + dichotomy⁰² の 2 等式から純代数で導く方法は, formalization で重い orthogonality 引数を完全に回避できる. m_k ≥ 1 (Perron 存在) と m_k ≤ 1 (Perron 一意性) を別個に証明するのではなく, 数式の交叉から `m_k (k² + 1) = k² + 1` を直接得る.

## Phase 2: Case B `srg_case_B` (commit `c3fd6b9`)

`IsSquare (4k - 3)` ⟹ ∃ u₀, `4k - 3 = u₀ * u₀`. `u := |u₀|` で u ≥ 0, `u² = 4k - 3`.

`k ≥ 1` から `4k - 3 ≥ 1` ⟹ `u ≥ 1`.

`Real.sqrt(4k - 3) = u` を `Real.sqrt_sq` で.

Disc equation + m_k = 1:
```
(m_+ - m_-) · u = k² + 1 - (2k + 1) = k(k - 2)  over ℝ
```

`exact_mod_cast` で ℤ に格上げ:
```
r * u = k(k - 2)   over ℤ,  r := (m_+ : ℤ) - (m_- : ℤ)
```

⟹ `u | k(k - 2)`.

中心 identity (commit `c3fd6b9` の linear_combination):
```
16 k(k - 2) = u⁴ - 2u² - 15      via u² = 4k - 3
```

導出: `(u² + 4k - 5)(u² - 4k + 3) = u⁴ - 2u² - 16k² + 32k - 15`,
で `u² - 4k + 3 = 0` から `(u² + 4k - 5) * 0 = 0` ⟹ `u⁴ - 2u² - 15 = 16 k(k - 2)`.

Lean では:
```lean
have h16 : 16 * ((k : ℤ) * ((k : ℤ) - 2)) = u * u * (u * u) - 2 * (u * u) - 15 := by
  have h := hu_sq
  linear_combination -(u * u + 4 * (k : ℤ) - 5) * h
```

`u | u⁴` (= u · (u·u·u)) + `u | 2u²` (= u · 2u) ⟹ `u | u⁴ - 2u²`.
`u | 16 k(k - 2)` (= 16 · u·r).
差: `u | (u⁴ - 2u²) - 16 k(k - 2) = 15`.

`Int.le_of_dvd` で `u ≤ 15`. `interval_cases u` で u ∈ {1, ..., 15} 列挙:
- u ∈ {1, 3, 5, 15} (15 の正の約数): `4k - 3 = u²` から k ∈ {1, 3, 7, 57} 計算.
- 残 11 ケース: u ∤ 15 を `decide` で矛盾.
