### `Foundations/LinearAlgebra/PowPrimeTrace.lean` (新規)

主結果: 任意の ℚ-線形自己準同型 `f : W →ₗ[ℚ] W` (有限次元) について
`f^p = 1` (p prime) なら `trace(f) ∈ ℤ`。

戦略:
- `cyclicProjection f p = (1/p) • (1 + f + f² + ... + f^(p-1))` を定義
- `s² = p·s` (cyclicSum_sq) より `cyclicProjection` は idempotent
- `range(proj) = ker(f - 1)` (固定部分空間), 上で `f` は恒等
- `ker(proj) = ker(cyclicSum f p) = ker(aeval f cyclotomic_p)` (p prime)
- DirectSum trace decomp 経由で `trace = dim(range) + trace(f|_{ker})`
- 上で既存 `trace_package_of_cyclotomic_prime_aeval_eq_zero` を適用:
  `dim(ker) = (p-1)·γ`, `trace(f|_{ker}) = -γ`
- 合計: `trace(f) = dim(range) - γ ∈ ℤ`

これは既存 `exists_int_trace_of_involutive` (p = 2) の **prime-order 一般化**。

```lean
-- 軌道 O が定める基点 a の「近傍」
def orbitalNeighborhood (O : orbital G Ω) (a : Ω) : Set Ω :=
  { c | Quotient.mk'' (a, c) = O }

-- G-equivariance bridge: c ∈ N(g•a) ↔ g⁻¹•c ∈ N(a)
theorem mem_orbitalNeighborhood_smul_left
    (O : orbital G Ω) (g : G) (a c : Ω) :
    c ∈ orbitalNeighborhood G Ω O (g • a) ↔
    g⁻¹ • c ∈ orbitalNeighborhood G Ω O a

-- 2つの軌道近傍の交わり数
noncomputable def orbitalIntersectionCount (O₁ O₂ : orbital G Ω) (a b : Ω) : ℕ :=
  Nat.card { c // c ∈ orbitalNeighborhood G Ω O₁ a ∧
                   c ∈ orbitalNeighborhood G Ω O₂ b }

-- G-invariance of the count under diagonal action
theorem orbitalIntersectionCount_smul (O₁ O₂ : orbital G Ω) (g : G) (a b : Ω) :
    orbitalIntersectionCount G Ω O₁ O₂ (g • a) (g • b) =
    orbitalIntersectionCount G Ω O₁ O₂ a b

-- 同じ orbital に属する (a,b) では count が一致 (paper-faithful constancy)
theorem orbitalIntersectionCount_orbital_invariant
    (O₁ O₂ : orbital G Ω) {a b a' b' : Ω}
    (h : SameOrbital G Ω (a, b) (a', b')) :
    orbitalIntersectionCount G Ω O₁ O₂ a b =
    orbitalIntersectionCount G Ω O₁ O₂ a' b'

-- Rank-3 predicate
def IsRank3 (G Ω : Type*) [Group G] [MulAction G Ω] : Prop :=
```
